打开路由转发
临时生效
 echo “1” > /proc/sys/net/ipv4/ip_forward
永久生效
 vim /etc/sysctl.conf
   net.ipv4.ip_forward=1
 执行sysctl Cp立即生效

添加静态路由
  route add Cnet 192.168.18.0 netmask 255.255.255.0 gw 192.168.31.100
  route add Cnet 192.168.18.0/24 gw 192.168.31.100
  route add Cnet 192.168.18.0/24 dev eth0
添加一条从本机到192.168.18.0/24网段经过192.168.31.100网关的路由

  vim /etc/sysconfig/network-scripts/route-eth0
    192.168.18.0/24 via 192.168.31.100
  Service network restart

  vim /etc/sysconfig/static-routes
    any net 192.168.18.0/24 gw 192.168.31.254
  service network restart

删除一条路由
 route del Cnet 192.168.18.0 netmask 255.255.255.0

/etc/sysconfig/network-scripts/下
创建 route-ens33
	1)10.0.0.0/24 via 192.168.10.1
	2)ADDRESS0=10.0.0.0
	    NETMASK=255.255.255.0
	    GATEWAY=192.168.10.1

在ifcfg-ens33的配置文件中添加
/usr/sbin/route  add -net 10.0.0.0/24  gw  192.0.0.254
ifdown和ifup网口也不会丢失

Flags U代表它是一个直连路由，相当于在一个局域网内
          UG表示需要经过网关
Gateway 0.0.0.0 意味着这是一条直连规则，即：凡是匹配到这条规则的包，直接经过本机的ens33网卡，通过二层网络直接发往目的主机
添加临时ip
ip a  add 10.0.0.1/24 dev ens33
删除临时ip，也可以删除永久ip，删除永久ip的时候会warning，但是也可以删除
ip a del 10.0.0.1/24 dev ens33
